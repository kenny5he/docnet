# [Eslint](https://zh-hans.eslint.org/docs/latest/)
## Eslint Useful (使用)
1. 安装Eslint
```shell
npm install --save-dev eslint
```
2. 安装Eslint-parser
```shell
npm install @babel/eslint-parser --save-dev
```
3. 初始化配置文件
```shell
npm init @eslint/config
```
