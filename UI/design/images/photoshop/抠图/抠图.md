# 抠图

## 魔棒工具 (快捷键 W)
1. 适用范围: 公司图标、logo
2. 前提条件：背景为白色，或背景为单一颜色
3. 图层(右侧工具栏) -> 解锁 
4. 移除背景
5. 连续 (图片主体中同颜色处是否跟随移除)
6. 容差 (默认为32，容差数值越大，选择范围越大，相当于颜色容错)


## 橡皮擦工具 (快捷键 E)
1. 适用范围: 公司图标、logo
2. 背景橡皮擦工具
   1. 先取样 (一次取样，取样本，即只删除样本相同位置背景)
3. 魔术橡皮擦工具
   1. 在纯色处点击，即自动擦除，无需解锁


## 快速选择工具 (快捷键 W)
1. 适用范围: 人像 (不过于复杂)
2. 选择图像主体
3. 选择移动覆盖主体内容区域
4. 头部工具栏 -> 选择并遮住 -> 右侧 属性(视图模式) -> 选择叠加 (红色代表未选中区域)
5. Ctrl + '+'号 放大、Ctrl + '-'号 缩小
6. Atl + '+' 放大区域、Alt + '-'号 减小区域
7. 左侧工具栏 笔形图标 -> 在多余处点击 (自动计算消除多余部分)
8. 右侧 属性(视图模式) -> 选择 图层 (透明图层) 查看区域 
9. 右侧 属性(视图模式) -> 选择 黑白 -> 全局调整 (平滑、羽化、对比度) -> 确定
10. 右下角 (日本旗图标) 添加蒙版 即完成抠图 

## 钢笔工具 (快捷键 P)
1. 适用范围: 自由度高、高阶使用
2. 图片放大，更精确
3. 头部工具栏 -> 设置(齿轮图标) -> 橡皮带(勾选) 即可预览路径 
4. 以点击建立 锚点(圈定内容起点) -> 点击圈定的下个
5. 按住鼠标拖动 (建立带幅度的弧线) -> 按住Ctrl键 控制手柄 
6. 按住Alt 键 单独控制某个 弧度的 手柄，从而控制弧度，确定圈定范围
7. 确定范围后，右下角 (日本旗图标) 添加蒙版 即完成抠图

## 通道抠图
1. 适用范围：主体和素材都较为复杂的图片
2. 选择右侧 图层旁的通道，选择对比度最高的通道
3. 复制对比度最高的通道，使主体与背景 颜色差异更大
4. 头部文字工具栏(图像) -> 应用图像 -> 混合模式调整为颜色加深
5. 头部文字工具栏(图像) -> 调整 -> 色阶(Ctrl + L) -> 输入色阶 左边往中间拖动(黑色部分更黑)、右边往中间拖动(白色部分更白)
6. 通道 -> 选择步骤3中复制的对比度较高的通道 -> 按住Ctrl点击(建立选区)
7. 头部文字工具栏(选择) -> 反选
8. 通道 -> 选择RGB模式
9. 图层 -> 右下角 (日本旗图标) 添加蒙版 即完成抠图
