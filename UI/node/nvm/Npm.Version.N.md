### 升级Node 版本
1. 先查看本机node.js版本
```
node -v
```
2. 清除node.js的cache
```
sudo npm cache clean -f
```
3. 安装 n 工具
```
sudo npm install -g n
```
4. 安装最新版本的node.js
```
sudo n stable
```
5. 查看本机的node.js版本
```
node -v
```
6. 更新npm到最新版
```
sudo npm install npm@latest -g
```
