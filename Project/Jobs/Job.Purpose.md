# 寻求工作职位 (己方)
1. 目的:
   1. 拓展业务方向
   2. 提升技术能力
   3. 巩固以往技术，达到以他人
   4. 提升管理能力（交流、表达能力、各方事业部协调处理能力）
   5. 寻找归属感(外包->正式)
   6. 提高与优秀的人的相识概率，使自己更优秀
2. 职位匹配度
   1. 公司主营业务？(挣钱方式)
   2. 公司技术导向？
      - 技术功底如何？(过去)
      - 未来投入？(未来)
   3. 职位与主营业务是否相关？
   4. 公司上级领导如何看待处理一个问题？
      1. 下属提出问题，是否为解决问题出发？或解决不了问题，解决提出问题的人？
      2. 公司是否有为解决问题提供资源，
         - 甚至为未来投资、不惜成本的魄力。(资金)
         - 甚至为解决问题扫清 旧有领导设置的阻碍。(权利)
      3. 上级是否为问题出谋划策？还是把交由自己处理？不闻不问，或只看结果？
   5. 公司如何对待员工的心声？是否包容、开放？
