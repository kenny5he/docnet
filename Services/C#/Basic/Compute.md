# 赋值运算符
| 运算符 | 描述                          | 案例 |
|-----|-----------------------------|----|
| =   | 简单赋值，计算右边表达式的值赋值给左边         |    |
| *=  | 复合赋值，a *= b 等价于 a = a * b   |    |
| /=  | 复合赋值，a /= b 等价于 a = a / b   |    |
| %=  | 复合赋值，a %= b 等价于 a = a % b   |    |
| +=  | 复合赋值，a += b 等价于 a = a + b   |    |
| -=  | 复合赋值，a -= b 等价于 a = a - b   |    |
| <<= | 复合赋值，a <<= b 等价于 a = a << b |    |
| >>= | 复合赋值，a >>= b 等价于 a = a >> b |    |
| &=  | 复合赋值，a &= b 等价于 a = a & b   |    |
| ^4= | 复合赋值，a ^= b 等价于 a = a ^ b   |    |
| \|= | 复合赋值，a \|= b 等价于 a = a \| b |    |
