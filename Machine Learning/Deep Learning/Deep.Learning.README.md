# 机器学习

### 深度学习：基本概念
1. 分类：学习方式

| 分类      | 数据                                                | 本质  | 应用  | 对比  |
|---------|---------------------------------------------------|-----|-----|-----|
| 有监督学习方式 | 有标签(有X有Y)                                         |     |     |     |
| 无监督学习方式 | 无标签(有X无Y)                                         |     |     |
| 半监督学习方式 |                                                   |     |     |
| 强化学习    | Environment环境和Agent智能体，环境和智能体交互，互动，反馈，学习调整策略的过程 |     |     | 

2. 有监督学习与无监督学习
    1. 监督学习：
        1. 目标明确
        2. 需要标签的训练数据
        3. 效果容易评估
    2. 无监督学习
        1. 目标不明确
        2. 不需要标签训练数据
        3. 效果不易评估
